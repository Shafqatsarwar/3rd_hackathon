# AGENTS.md Generator - Reference Documentation

## Overview
The AGENTS.md generator creates a comprehensive guide that helps AI agents understand repository structure, conventions, and guidelines.

## AGENTS.md Format
The generated file follows this structure:

```
# Repository Name - AGENTS.md

## Overview
Brief description of the repository's purpose and main components.

## Directory Structure
Detailed breakdown of the repository layout with key files highlighted.

## Technologies Used
List of programming languages, frameworks, and tools used in the project.

## Entry Points
Key files and functions that serve as main entry points for the application.

## Conventions
Coding standards, naming conventions, and architectural patterns used.

## Key Dependencies
Important external libraries and their roles in the system.

## Common Tasks
Frequently performed operations and how to execute them.
```

## Configuration Options
The generator can be configured with:
- Custom repository metadata
- Exclusion patterns for sensitive files
- Language-specific documentation templates
- Special handling for different project types

## Advanced Usage
For complex repositories, the generator supports:
- Multiple AGENTS.md files for different modules
- Integration with documentation systems
- Automated updates on structural changes
- Custom metadata extraction
