# AGENTS.md Generator

## When to Use
- When starting work on a new repository
- When repository structure has changed significantly
- When AI agents need to understand codebase organization

## Instructions
1. Run the generator script: `./scripts/generate_agents_md.sh`
2. Review the generated AGENTS.md file
3. Update as needed for your specific repository

## Validation
- [ ] AGENTS.md file exists in repository root
- [ ] File contains accurate directory structure
- [ ] Programming languages are correctly identified
- [ ] Key entry points are documented

See [REFERENCE.md](./REFERENCE.md) for advanced configuration options.
